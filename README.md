# IsLibrary

Image Processing & Recognition Standard Library  


[Requirements]  
1. Cmake >= 3.14.6  
2. C++20  
3. GCC Compiler for GoogleTest build.
4. MSVC or GCC or XCode Compiler  
5. Eigen 3.4.0  
6. GoogleTest 1.12.0  

[Features]  
1. NdArray based image memory.  
2. Image filtering.
